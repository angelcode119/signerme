# ⚠️ مهم: توکن ربات دوم

## 🔴 مشکل فعلی:

ربات دوم از همان توکن ربات اول استفاده می‌کند!
این باعث می‌شود که هر دو ربات به هم گیر بدهند.

## ✅ راه حل:

### 1️⃣ ساخت ربات جدید در تلگرام:

```
1. به @BotFather در تلگرام بروید
2. دستور /newbot را بفرستید
3. نام ربات را وارد کنید (مثلا: APK Analyzer)
4. username ربات را وارد کنید (مثلا: my_apk_analyzer_bot)
5. توکن جدید را کپی کنید
```

### 2️⃣ تغییر توکن در bot2.py:

فایل `bot2.py` را باز کنید و خط زیر را پیدا کنید:

```python
BOT2_TOKEN = BOT_TOKEN  # Using same token for now - CHANGE THIS!
```

و آن را به این تغییر دهید:

```python
BOT2_TOKEN = 'YOUR_NEW_TOKEN_HERE'  # توکن جدید از BotFather
```

مثال:
```python
BOT2_TOKEN = '1234567890:ABCdefGHIjklMNOpqrsTUVwxyz'
```

### 3️⃣ ریستارت ربات‌ها:

```bash
# توقف ربات‌ها (Ctrl+C)
# اجرای دوباره
python run_bots.py
```

## 📝 نکات:

- ✅ هر ربات باید توکن مستقل داشته باشد
- ✅ توکن‌ها را محرمانه نگه دارید
- ✅ هرگز توکن را commit نکنید در git
