## ğŸ‘¤ Account Management

### Single Session System
- Only one device can be logged in
- Logging in from new device logs out previous device
- You'll receive notification if logged out
- Security feature to protect your account

### Logout Process
1. Send `/logout` command
2. Confirm logout action
3. Session ended
4. Send `/start` to login again

**Why Logout?**
- Switching devices
- Security concerns
- Sharing device
- Account protection

## âŒ Troubleshooting

### Common Issues

**"Authentication failed"**
- Check your username
- Request new OTP
- Try again

**"APK file not found"**
- Contact admin
- APK might be removed
- Try different APK

**"Queue full"**
- Wait for current builds to finish
- Usually takes 1-2 minutes
- You'll be notified when ready

**"Build failed"**
- Try again
- Use Quick build instead
- Contact support if persists

**"Account banned"**
- Contact administrator
- Check ban reason in admin panel
- Appeal if mistake

### Error Messages

| Error | Solution |
|-------|----------|
| "Please login first" | Send `/start` |
| "Already generating" | Wait for current build |
| "Invalid code" | Enter 6-digit OTP |
| "User not found" | Check username |
| "No apps available" | Contact admin |
